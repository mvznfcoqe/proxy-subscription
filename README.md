# Proxy Subscription

Telegram Bot that gives out free proxies for subscribing to my channel

Live instance: @freecamsub_bot

## Local Development

1. Install Devcontainers plugin for your editor
2. Setup .env files from .env.example for each of app
3. Start development server with bun dev in each app folder

## Deployment

1. Fill `.env` file from `.env.example` in deployment folder
2. Run `docker compose up -d`
