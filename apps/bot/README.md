# Bot

Telegram bot built with Grammy, Bun, and Zod.

## Setup

1. Install dependencies:

```bash
bun install
```

2. Copy `.env.example` to `.env` and add your bot token:

```bash
cp .env.example .env
```

3. Get your bot token from [@BotFather](https://t.me/botfather) on Telegram and add it to `.env`

## Development

Run the bot in development mode with hot reload:

```bash
bun run dev
```

## Production

Run the bot in production mode:

```bash
bun run start
```
